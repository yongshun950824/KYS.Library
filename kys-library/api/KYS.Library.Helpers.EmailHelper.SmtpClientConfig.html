Remarks

This class encapsulates SMTP configuration options such as host, port, credentials, and SSL settings. It also provides an implicit conversion to SmtpClient for convenient use.

Example usage:

var smtpConfig = new SmtpClientConfig(
    host: "smtp.gmail.com",
    port: 587,
    senderEmail: "example@gmail.com",
    senderPassword: "password",
    enableSSL: true);

using var smtpClient = (SmtpClient)smtpConfig;
smtpClient.Send("example@gmail.com", "recipient@example.com", "Subject", "Body");
