BuildAndExpression<T>(List<FilterCriteria<T>>)

Builds a composite Expression<TDelegate> that represents a logical AND operation across multiple filter conditions.

public static Expression<Func<T, bool>> BuildAndExpression<T>(List<FilterCriteria<T>> filterCriterias) where T : IComparable

Parameters

filterCriterias List<FilterCriteria<T>>

A list of FilterCriteria<T> objects that define the individual comparison expressions to be combined with logical AND.

Returns

Expression<Func<T, bool>>

An Expression<TDelegate> of type Func<T, TResult> that evaluates to true only if all the filter conditions are satisfied. If filterCriterias is empty, the method returns an expression that always evaluates to true.

Type Parameters

T

The type of the object being filtered. Must implement IComparable.
