public static Task<ViewResult> ToViewResultAsync<T>(HttpContext httpContext, HttpResponseMessage response, string viewName, string errorViewName = "/Views/Shared/Error.cshtml")

Parameters

httpContext HttpContext

The current HTTP context, used to initialize the ViewDataDictionary.

response HttpResponseMessage

The raw response received from an API call.

viewName string

The name or path of the view to render on a successful API response.

errorViewName string

The name or path of the view to render if the API returns a non-success status code. Defaults to the Shared Error view.

Returns

Task<ViewResult>

A ViewResult containing an ApiReponseModel<T> as its model.

Type Parameters

T

The expected model type to be deserialized from the response body.

Remarks

This method automatically handles:

  • Checking IsSuccessStatusCode.
  • Asynchronous reading of the response content.
  • Deserialization of JSON content into type T.
  • Mapping API status codes and reason phrases to the view model.
